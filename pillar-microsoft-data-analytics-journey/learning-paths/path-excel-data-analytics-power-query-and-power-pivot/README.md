# Excel Data Analytics: Power Query and Power Pivot

**Learning Path within the Microsoft Data Analytics Journey Pillar**

## Overview

This learning path teaches Excel analysts how to transform manual data workflows into automated analytics pipelines using Power Query and Power Pivot. All while staying in Excel.

**Target Audience**: Business and data analysts who are Excel-proficient but manually clean and analyze data, ready to automate workflows.

**Duration**: 4-5 hours across 1 comprehensive course

**Status**: Draft (pending SME validation)

---

## What You'll Learn

- Automate data ingestion from SQL databases, CSV files, and folder sources
- Build reusable transformation logic using Power Query and M language
- Design analytical data models with Power Pivot and DAX measures
- Deliver automated, refreshable Excel dashboards reducing reporting time by 60-80%
