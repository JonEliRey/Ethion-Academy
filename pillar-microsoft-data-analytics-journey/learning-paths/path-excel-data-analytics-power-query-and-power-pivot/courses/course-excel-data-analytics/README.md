# Excel Data Analytics Course

**Transform manual Excel workflows into automated analytics pipelines**

## What You'll Build

A production-ready Excel analytics workbook that:
- Automatically ingests data from multiple sources (SQL + CSV)
- Applies cleaning and transformation logic via Power Query
- Models data with star schema relationships and DAX measures
- Delivers refreshed executive dashboards in under 5 minutes

---

## Course Structure

### Module 1: Modern Excel Setup for Analytics (45 min)
Set up Power Query and Power Pivot, create reusable workspace templates, understand the automation mindset.

**Lessons**: 4 lessons covering orientation, tool enablement, workspace design, and mindset shift

### Module 2: Power Query Data Ingestion (55 min)
Master data ingestion patterns: SQL connections, CSV/folder sources, parameterization, data quality checks.

**Lessons**: 4 lessons covering SQL connections, folder queries, parameters, and profiling

### Module 3: Power Query Transformation Patterns (60 min)
Build advanced transformations: merges, custom functions, error handling, refresh automation.

**Lessons**: 4 lessons covering sales shaping, purchasing normalization, M functions, and refresh automation

### Module 4: Power Pivot Modeling & Analysis (55 min)
Design analytical models: star schema relationships, DAX measures, executive-ready pivot tables.

**Lessons**: 4 lessons covering model loading, relationships, DAX measures, and pivot visualizations

### Module 5: Capstone - WWI Analytics Sprint (60 min)
Apply all concepts in comprehensive project: multi-source pipeline, Power Pivot model, automated dashboard.

**Lessons**: 4 lessons covering capstone brief, pipeline build, modeling, and final delivery with self-assessment

## Assessment

**Self-Directed Capstone Project** covering:
- Data ingestion from multiple WWI sources
- Power Query transformations and error handling
- Power Pivot model with relationships and DAX measures

**Rubric Categories** (scored 0-3 each, 15 points total):
- Data Ingestion Quality
- Transformation Completeness
- Data Model Design
- Insights & Narrative
- Refresh Reliability

**Passing Score**: 12/15 (80%)
