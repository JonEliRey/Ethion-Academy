# Excel Data Analytics: Power Query and Power Pivot - Learning Path

Complete learning path for mastering Excel data analytics through Power Query and Power Pivot features.

## Purpose

Transform from basic Excel user to proficient data analyst. This path teaches modern Excel analytics capabilities used in professional business intelligence and data analysis roles.

## Overview

**Target Audience**: Business and data analysts familiar with basic Excel who want to level up to advanced analytics

**Duration**: 12-15 hours across 5 courses

**Format**: 70+ video lessons (8-15 minutes each), hands-on exercises with World Wide Importers dataset

## Learning Outcomes

By completing this path, you will:
- ✅ Connect to and import data from multiple sources
- ✅ Transform and clean data automatically with Power Query
- ✅ Build multi-table data models with proper relationships
- ✅ Write DAX formulas for business calculations
- ✅ Create interactive analytical dashboards
- ✅ Apply industry best practices for Excel analytics

## Prerequisites

- Basic Excel skills (formulas, worksheets, functions like SUM/AVERAGE)
- Understanding of business data concepts
- Microsoft Excel 2019 or newer (365 recommended)
- No programming experience required

## Course Structure

### Course 1: Excel Data Foundations (2-3 hours)
Foundation in proper data structures before diving into advanced features.

**Key Topics**: Data types, Excel tables, data validation, essential formulas

### Course 2: Power Query Fundamentals (3-4 hours)
Master Power Query interface and core transformation operations.

**Key Topics**: Connecting to sources, basic transformations, column operations, query management

### Course 3: Advanced Power Query (2-3 hours)
Advanced techniques for complex data scenarios.

**Key Topics**: Combining queries, custom columns, parameters, error handling

### Course 4: Power Pivot and Data Modeling (3-4 hours)
Build data models and learn DAX for advanced calculations.

**Key Topics**: Relationships, DAX fundamentals, calculated columns vs measures, time intelligence

### Course 5: Analytics and Reporting (2-3 hours)
Create interactive dashboards and learn best practices.

**Key Topics**: PivotTables with data models, slicers, dashboard design, automation basics
