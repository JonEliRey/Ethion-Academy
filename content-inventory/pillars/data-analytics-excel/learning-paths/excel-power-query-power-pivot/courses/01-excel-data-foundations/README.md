# Course 1: Excel Data Foundations

## Overview

**Duration**: 2-3 hours  
**Level**: Beginner  
**Prerequisites**: Basic Excel familiarity

Establish strong foundations in Excel data structures and basic analysis before diving into Power Query and Power Pivot.

## Purpose

Many Excel users jump straight to advanced features without understanding proper data structure. This course ensures you have a solid foundation in data best practices, Excel tables, and essential formulas that will make your Power Query and Power Pivot work much more effective.

## Learning Outcomes

By completing this course, you will:
- Understand how Excel stores and interprets different data types
- Structure data properly for analysis (flat vs. hierarchical)
- Use Excel tables for dynamic, structured data ranges
- Implement data validation to ensure data quality
- Apply essential analytical formulas with confidence

## Key Concepts

### Data Structure Best Practices
- One row per record (flat structure)
- Column headers in first row only
- No blank rows or columns within data
- Consistent data types within columns
- No merged cells in data range

### Excel Tables Benefits
- Automatic formula filling
- Dynamic ranges (expand/contract automatically)
- Structured references (readable formulas)
- Easy filtering and sorting
- Professional appearance

### Data Validation Use Cases
- Ensure consistent entry (drop-downs)
- Prevent errors (numeric ranges)
- Control data quality at input
- Create user-friendly forms
