# Data Analytics with Excel - Pillar

This pillar contains learning content focused on mastering data analytics using Microsoft Excel's advanced features, particularly Power Query and Power Pivot.

## Purpose

Transform Excel users into proficient data analysts by teaching modern Excel analytics capabilities. Content is designed for business and data analysts who want to unlock Excel's full analytical power without requiring programming skills.

- `pillar-data-analytics-excel.md` - Complete pillar specification including scope, learning objectives, and business context

- `learning-paths/excel-power-query-power-pivot/` - Comprehensive journey through Excel data analytics

## Target Audience

- **Roles**: Business Analyst, Data Analyst, Financial Analyst
- **Experience Level**: Beginner to Intermediate
- **Prerequisites**: Basic Excel knowledge (formulas, worksheets, basic functions)

## Key Topics

- Data connection and import
- Data transformation with Power Query
- Data modeling with Power Pivot
- DAX (Data Analysis Expressions)
- Interactive dashboards and reporting
- Analytics best practices

## Learning Outcomes

By completing this pillar, learners will be able to:
- Extract and transform data from multiple sources using Power Query
- Build relational data models using Power Pivot
- Write DAX formulas for business calculations
- Create interactive analytical dashboards
- Apply best practices for maintainable Excel analytics solutions
